# HamburgerMenu

A collapsible menu component useful for mobile navigation or off-canvas menus.

## Props

| Prop | Type | Description |
|---|---|---|
| `children` | `ReactNode` | Content to display inside the menu. |
| `position` | `'left' \| 'right'` | Position of the menu. Default: `'right'`. |
| `label` | `string` | Accessibility label for the toggle button. Default: `'Toggle menu'`. |
| `className` | `string` | Optional CSS class name for the wrapper. |

## JSON Data Example

```json
{
  "type": "HamburgerMenu",
  "data": {
    "id": "hamburger-default",
    "className": "relative bg-primary text-primary-950",
    "position": "right",
    "label": "Toggle navigation menu",
    "children": [
      {
        "type": "MenuGridItem",
        "data": {
          "id": "menu-grid-item-compact",
          "title": "Mobile App",
          "href": "#mobile",
          "icon": "mdi:cellphone",
          "compact": true,
          "className": "bg-primary text-primary-950 shadow-md w-48 m-6"
        }
      }
    ]
  }
}
```
